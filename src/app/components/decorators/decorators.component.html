Decorators Work!

What are decorators?

Decorates refers to a piece of code, that denote the purpose of a class, element , method ie upon which its placed.

type of decorators.

Class decorators : decorator that place up on a class ex: @Component(use upon component class), @ Pipe(for pipe class), @Injectable(use upon service class), Directive , Module.

Property decorator : ex: @Input, @Output, @HostBinding.

Method Decorator : decorator use upon method ex : @HostListener. ex : @HostListener('onScroll') onScroll() method.

Paramenter Decorator : ex: Inject(HttpClient) here inject is paramenter decorator.
